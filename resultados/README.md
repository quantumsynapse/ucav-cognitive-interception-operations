# Resultados

Os arquivos desta pasta são gerados por:

```bash
make referencia
make run
```

A configuração padrão executa 1.000 cenários por estratégia.

Arquivos gerados:

- `trajetoria_referencia.csv`;
- `dialogo_referencia.txt`;
- `resumo_referencia.txt`;
- `resumo_monte_carlo.txt`.

Os resultados são sintéticos e não devem ser interpretados como desempenho de uma aeronave real.
